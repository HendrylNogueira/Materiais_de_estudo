Código fonte

Vamos ver tags que deixam a visualização mais agradável, assim como códigos de programas.

Vamos utilizar a Tag code

n = int(input('Digite um número: ')) if n % 2 == 0: print('Esse é um número par.') else: print('Esse número é ímpar.')

Deixando o código mais bonito e da ordem correta com a tag 'pre'

        
    n = int(input('Digite um número: '))
    if n % 2 == 0:
        print('Esse é um número par.')
    else:
        print('Esse número é ímpar.')
